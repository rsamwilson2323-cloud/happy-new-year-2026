# 🎆 Happy New Year 2026

A beautiful and interactive **New Year celebration animation** built with **HTML5 Canvas, CSS, and JavaScript**. The project displays animated fireworks, glowing text effects, floating balloons, and festive visual effects to celebrate the arrival of 2026.

## ✨ Features

* 🎇 Animated firework effects
* ✨ Glowing "Happy New Year 2026" text
* 🎈 Floating balloon animations
* 🌙 Smooth particle and lighting effects
* 📱 Responsive fullscreen design
* ⚡ Pure HTML, CSS, and JavaScript
* 🎨 Canvas-based rendering for smooth performance

### Clone the Repository

```bash

```
